Проект №3 - WEB. Название проекта - "CityAssistant"

Постановка задачи:
	- Создать приложение, в котором пользователи могли бы добавлять известные им проблемы городской среды, выбирая категорию и локацию,
	а пользователи с правами менеджера могли бы просматривать задачи, удалять их или отмечать как выполненные или не выполненные

Запуск приложения:
	Из сторонних библиотек требуется только flask, flask-wtf, flask-login, flask-sqlalchemy.
	Запускать файл main.py.

Функционал приложения:
	- Реализована регистрация пользователя и авторизация
	- Пользователь может зарегистрироваться только с правами категории user

	ВАЖНО!:
	- Админские права (роль manager) есть у пользователя: логин: overlord@mail.ru, пароль: overlord. Используйте эти данные для проверки работы.

	- логины и пароли других пользователей можно посмотреть в БД
	- "Обычный" пользователь может только добавлять задачи и просматривать их список на главной странице
	- У имеющихся в БД на данный момент задач стоят координаты Парижа.
	- Менеджер не может добавлять задачи (защита от накрутки), может просматривать подробную информацию о задаче (включая геолокацию через статические яндекс-карты),
	 удалять задачи, помечать их как выполненные или не выполненные. (Кнопка "Выполнено/не выполнено", меняется в зависимости от статуса)
	- У "выполненных" задач на главной странице появляется соответствующая подпись и зеленый фон
	
	
	- Не удалось реализовать более крутой виджет яндекс-карт, с которым можно взаимодействовать, крутить, менять масштаб и т.д.,
	так как реализован он на Javascript, и для того, чтобы довести этот функционал до ума, требуется больше времени
