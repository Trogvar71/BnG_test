Проект №2- PyGame. Название проекта - "NetAnalyzer"

Постановка задачи:
	- Написать прототип игры в одном из моих любимых жанров - Tower Defence - и реализовать все основные механики жанра.


Запуск приложения:
	Из сторонних библиотек требуется только pygame. Модули sys, math и subprocess входят в стандартную библиотеку, модули enemies, towersr и projectiles самодельные.
	Запускать файл main.py.

Описание игры и реализованных механик:
	- при запуске игры появляется стартовое окно
	- Нажимаем "Начать"
	- появляется уровень.
		Уровни и вся их информация считываются из файлов, что облегчает добавление новых - просто пишем новый файл уровня, и он будет грузиться в игру сам.
	- бежевые плитки - маршрут врагов
	- зеленые плитки - просто трава, окружение
	- клики по серым плиткам позволяют построить на них башню
	- Ярко синий квадрат - это портал. Достигая его, враги отнимают у вас жизнь, туда их пускать нельзя
	- первая волна начнется через 10 секунд после запуска уровня. На первом уровне три волны.
	- Чтобы сдержать армии тьмы, вам нужно строить башни на серых клетках (в данной версии один вид башен)
		Башня имеет параметры урона, скорострельности, радиуса и так далее. От класса Tower дальше его отнаследовать
		новые типы башен, если нужно
	- Начинают идти враги.
		У врагов есть скорость, здоровье, интервал появления. От класса Enemy легко отнаследовать новые типы врагов
		со своими уникальными особенностями
	- У башни намеренно оставлен большой радиус стрельбы, так что даже одна башня справится с первым уровнем 
		(а еще есть пасхалка, позволяющая ускорить процесс уничтожения врагов)
	- башня стоит 50 золотых и строится только при наличии нужной суммы
	- Если в портал пройдет 10 врагов, появится окно конца игры (то есть в игре есть условие поражения)
	- Если вы выстоите первый уровень, сохранив хотя бы одну жизнь, автоматически появится следующий уровень (сейчас их два)
	- Если вы пройдете все уровни, игра вернется на стартовое окно с кнопкой "начать"
		Подсказка: второй уровень с одной башней уже не пройти
